Шаг 1: Подготовка.

Перед установкой Windows убедитесь, что у вас есть флешка минимум на 8 ГБ. Если её нет купите.

Шаг 2: Установка утилиты для создания загрузочной флешки.

Скачайте с официального сайта Microsoft утилиту Media Creation Tool

Шаг 3: Настройка Media Creation Tool.

Вставляем флешку. После запуска Media Creation Tool. Вам предстоит выбрать версию Windows и язык будущей системы. Нажимаете далее.

Шаг 4: Настройки для создания загрузочной флешки.

Далее Вам предложит выбрать USB-устройство или ISO файл. Выбираете USB-устройство. Нажимаете далее.

Шаг 5: Процесс создания загрузочной флешки.

Теперь лишь требуется ожидать завершения процесса создания загрузочной флешки.

Шаг 6: Перезагрузка с зажатой клавишей "Shift".

Далее надо в меню пуск выбрать перезагрузка и зажать клавишу "Shift". Но в том случае, если мы переустанавливаем Windows.

Иначе же нам нужно зайти в биос и выбрать флешку приоритнее диска. Для каждой материнской платы по своему заход в биос. Посмотрите для своего в интернете

Шаг 7: Меню выбора действия.

Выбираем использовать устройство (USB-устройство)

Далее в списке ищем нашу флешку и выбираем её.

Шаг 8: Установка Windows.

Выбираем устанавливаемый язык. Формат времени и денежных единиц. Метод ввода. Потом нажимаем кнопку "Далее".

Шаг 9: Подтверждение установки.

Нажимаем кнопку установить

и начинается установка.

Шаг 10: Активация Windows.

Далее появляется окно с предложением ввести ключ продукта. Это можно сделать позже. Нажмите у меня нет ключа продукта

Шаг 11: Выбор редакции Windows.

Выбирайте Windows 11 Домашняя или Windows 11 Pro. В другой статье Вы сможете узнать об различиях в редакциях.

Шаг 12: Лицензионное соглашение Microsoft.

Далее Вам надо принять лицензионное соглашение Microsoft. Поставьте галочку и нажмите "Далее". По желанию можно прочитать.

Шаг 13: Выбор типа установки системы.

Так как установка системы идет с нуля. В этом окне надо выбрать пункт "Выборочная установка".

Шаг 14: Разметка дисков.

Если у вас единственный диск. Выбираем его. Нажимаем "Создать". Также по желанию один физический диск можно разбить на 2 логических раздела. Нажимаем "Далее".

Шаг 15: Процесс установки Windows.

Здесь требуется ожидание в зависимости от вида накопителя на который устанавливает операционная система Windows.

Скорость установки может отличаться. На SSD установка идет примерно 5 минут. На HDD от 10 до 20 минут

Шаг 16: Перезагрузка.

После завершения установки. Произойдет автоматическая перезагрузка. Флешку надо достать.

Далее система предложит её настроить. Настраивайте её на свой вкус.
